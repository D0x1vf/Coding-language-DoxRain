## üìñ Language Guide

### Basic Syntax
```doxrain
//log//dox
# Variables and output
-var message ~ "Hello, Professional World!"
-Show message

# Functions
Funcxd calculate(x, y) {
    -var result ~ x + y
    -Show "Result: " + result
}

Call calculate(10, 20)
//term//dox
```

### Web Development
```doxrain
//log//dox
WebServer 3000 {
    Route "/api/security" {
        -Show "Security API endpoint"
    }
    Route "/api/ai" {
        -Show "AI processing endpoint"
    }
}
-Show "Professional server running on port 3000"
//term//dox
```

### Object-Oriented Programming
```doxrain
//log//dox
Classxd CyberTool {
    Methodxd constructor(name) {
        -var tool_name ~ name
        -Show "Cybersecurity tool initialized"
    }
    
    Methodxd scan(target) {
        -Show "Scanning " + target
        -Show "Security analysis complete"
    }
}

# Create instance
-var scanner ~ CyberTool("Advanced Scanner")
Call scanner.scan("192.168.1.1")
//term//dox
```
